Diketahui Celsius

K = C + 273.15
F = C × 9/5 + 32
K = C × 4/5

Diketahui Fahrenheit

C = (F - 32) × 5/9
K = (F + 459.67) × 5/9
R = (F - 32) × 4/9

Diketahui Kelvin

C = K - 273.15
F = (K × 9/5) - 459.67
R = (K - 273.15) × 4/5

Diketahui Reamur

C = 5/4 × R
F = 9/4 × R +32
K = 5/4 × R + 273.15
